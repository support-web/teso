# Palmistry Gate ランディングページ - 統合コーディング依頼書

このファイルは、生成AIに一括で渡すための統合プロンプトです。
より細かい調整が必要な場合は、個別のセクションファイルを使用してください。

---

## プロジェクト概要

手相占いサービス「Palmistry Gate」のランディングページを作成してください。
神秘的で高級感のある和モダンデザインです。

## 技術スタック
- HTML5 + CSS3（モダンCSS使用）
- バニラJavaScript（必要最小限）
- レスポンシブ対応必須（Desktop / Tablet / Mobile）

## デザイントーン・キーワード
- 神秘的（Mystical）
- 高級感（Luxurious）
- 和モダン（Japanese Modern）
- エレガント（Elegant）
- 繊細（Delicate）
- 余白を活かした（Spacious）

## セクション構成と詳細仕様

### 1. ヘッダー（固定ナビゲーション）

**レイアウト:**
- position: fixed, width: 100%, height: 70px
- 背景: rgba(10, 26, 22, 0.95) + backdrop-filter: blur(10px)
- 下部ボーダー: 1px solid rgba(201, 162, 39, 0.1)

**コンテンツ:**
- 左: ロゴ（手のアイコンSVG + "Palmistry Gate"テキスト）
  - フォント: Cormorant Garamond, 22px, font-weight: 500
- 中央: ナビメニュー（ホーム、手相の鑑定、プロ占い師、HAN、ブログイン）
  - フォント: Noto Sans JP, 13px, color: var(--text-secondary)
  - ホバー時: color: var(--text-primary)
- 右: 「オンライン対面」ボタン
  - ゴールドボーダー、pill型（border-radius: 20px）
  - ホバー時: 背景ゴールド、文字ダーク

---

### 2. ヒーローセクション

**レイアウト:**
- 高さ: 100vh（最小800px）
- 2カラム: 左55%（テキスト）/ 右45%（画像）
- padding-top: 70px（ヘッダー分）

**背景装飾:**
- ダークグリーン背景の上にSVGで描画
- ゴールドの曲線ライン3本（異なる透過度: 0.25, 0.3, 0.5）
- 光の粒子（小さな円）を10個程度配置
- stroke-width: 1〜2px

**キャッチコピー:**
```
あなたの掌には、
まだ知らない可能性が
刻まれている。
```
- フォント: Noto Serif JP, 42px, font-weight: 500
- line-height: 1.8, letter-spacing: 0.12em
- color: #ffffff

**CTAボタン:**
- テキスト: 「無料でAI手相診断を試す ▶」
- 背景: transparent
- ボーダー: 1px solid var(--accent-gold)
- border-radius: 50px（pill型）
- padding: 16px 40px
- フォント: Noto Sans JP, 15px
- ホバー時: 背景ゴールド、文字#0a1a16

**右側画像:**
- 画像がない場合はプレースホルダーを配置
- 推奨サイズ: 800 x 1000px
- dashed border, 角丸20px

---

### 3. サービスカードセクション（3カラム）

**レイアウト:**
- 3カラム均等配置（grid）
- gap: 24px
- max-width: 1100px, margin: 0 auto
- padding: 80px 40px

**カード共通スタイル:**
```css
background: linear-gradient(180deg, rgba(201, 162, 39, 0.08) 0%, rgba(201, 162, 39, 0.02) 100%);
border: 1px solid rgba(201, 162, 39, 0.2);
border-radius: 8px;
padding: 40px 32px;
text-align: center;
```

**ホバー時:**
- transform: translateY(-4px)
- border-color: rgba(201, 162, 39, 0.4)
- 上部にゴールドのアクセントライン表示

**カード内容:**

| カード | タイトル | アイコン概要 |
|--------|----------|--------------|
| 1 | AIスピード診断 | スマホ + AI波形 |
| 2 | プロ占い師に相談 | 人物 + 星の装飾 |
| 3 | オンライン対面鑑定 | モニター + カメラ + 人影 |

**タイトル:**
- フォント: Noto Sans JP, 16px, font-weight: 500
- color: var(--text-primary)

**説明文:**
- フォント: Noto Sans JP, 13px, font-weight: 300
- color: var(--text-secondary)
- line-height: 1.8

```css
  /* ヒーロー: 1カラム縦積み */

  /* ナビ: ハンバーガーメニュー */
}
```

## 出力要件

1. **ファイル形式:** 単一のindex.html（CSSは`<style>`タグ内に記述）
2. **コメント:** 各セクションの開始位置にコメントを記載
3. **画像:** プレースホルダーで対応（後から差し替え可能な構造）
4. **アイコン:** SVGをインラインで埋め込み
5. **アニメーション:** CSS transitionとkeyframesで実装

## 追加の注意事項

1. **CSS変数を活用** - 色やフォントは必ずCSS変数で定義
2. **セマンティックHTML** - header, section, nav等を適切に使用
3. **アクセシビリティ** - alt属性、適切なコントラスト比
4. **パフォーマンス** - 不要なライブラリは使用しない
